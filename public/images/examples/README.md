# Example Images

This folder should contain 6 images total:

## English Images (shown on English page)
- `friendly-en.jpg` - Friendly tone example card
- `motivational-en.jpg` - Motivational tone example card
- `reflective-en.jpg` - Reflective tone example card

## French Images (shown on French page)
- `friendly-fr.jpg` - Friendly tone example card (French)
- `motivational-fr.jpg` - Motivational tone example card (French)
- `reflective-fr.jpg` - Reflective tone example card (French)

## Image Specifications
- **Recommended aspect ratio**: 4:5 (portrait orientation)
- **Recommended size**: 800x1000px or similar
- **Format**: JPG, PNG, or WebP
- **Optimization**: Compress images before adding to keep file sizes reasonable

The component will automatically display:
- 3 English cards when the page language is set to English
- 3 French cards when the page language is set to French
